Литье полипропиленовой пленки

Одно из основных направлений деятельности ООО "Вектор-Н" является производство полипропиленовой пленки CPP (CAST), пригодной для упаковки пищевых продуктов и соответствующей требованиям Технического регламента Таможенного Союза ТР ТС 005/2011 "О безопасности упаковки". Декларация соответствия № ТС RU Д-RU.АЯ24.В.00900.

Для производства пленки используется только первичное сырьё высшего качества ведущих российских нефтеперерабатывающих компаний.

Области применения пленки CPP (CAST):

  • Для упаковки хлебо-булочных и кондитерских изделий.
  • Для упаковки овощей.
  • Для упаковки зелени.
  • Для упаковки салфеток бумажных, полотенец и туалетной бумаги.
  • Для упаковки текстильных изделий.
  • Для упаковки бакалейных товаров.
  • Для упаковки моющих средств.

Кроме того у нас Вы можете приобрести BOPP пленку, жемчужную пленку, металлизированную пленку производства Биаксплен.
